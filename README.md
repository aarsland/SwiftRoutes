## Requirements

iOS 8.0 or later.

## Installation

### Carthage

```ruby
github "takecian/SwiftRoutes"
```

### Cocoapods

SwiftRoutes is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "SwiftRoutes"
```
